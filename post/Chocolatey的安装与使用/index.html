Chocolatey的安装与使用

一、Chocolatey简介

Chocolatey是Windows平台上一款软件包管理器，便于安装各种软件包，用命令行的方式完成安装，并省去了环境变量设置等操作

二、Chocolatey安装

  1. 首先以管理员的身份打开PowerShell
iex ((new-object net.webclient).DownloadString('https://chocolatey.org/install.ps1'))

# 或者下面的命令
iwr https://cin.st | iex

以上安装默认安装到C:\ProgramData\Chocolatey

如果想要安装到其他文件下，可以先新建环境变量ChocolateyInstall，值是希望安装的文件夹路径。然后再运行以上命令即可

  1. 检查是否安装成功
# 查看choco版本号
choco -v

二、Chocolatey安装软件操作

# 安装软件包
choco install 软件包名称

#自定义安装路径
choco install 软件包名称 --install-directory=路径

命令还可添加-y表示默认同意安装示例：

choco install Hugo --install-directory=D:\aaaa -y

三、Chocolatey其他操作

  • search命令
choco search 关键字

会搜索出所有包含关键字的软件包。例如搜索jdk会列出jdk相关软件

  • update命令
choco upgrade 软件包名称

更新软件包

  • uninstall命令
choco uninstall 软件包名称
