Hugo博客搭建

一、安装Hugo（基于Windows）

首先在powershell中安装choco包管理器，便于Hugo的安装

powershell -NoProfile -ExecutionPolicy unrestricted -Command "iex ((new-object net.webclient).DownloadString('https://chocolatey.org/install.ps1'))"

# 设置环境变量
SET PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin

安装完成后可以通过choco -version命令查看choco版本号

然后安装Hugo

choco install hugo -confirm

#查看Hugo是否安装成功
hugo -version

二、使用Hugo创建博客

到合适的路径

使用hugo new site 创建博客（创建新的文件夹）

hugo new site myblog
# myblog为自己博客项目名字

myblog文件夹结构如下：

├── archetypes
│   └── default.md
├── config.toml         # 博客站点的配置文件
├── content             # 博客文章所在目录
├── data                
├── layouts             # 网站布局
├── static              # 一些静态内容
└── themes              # 博客主题

三、下载主题

到Hugo主题网站挑选主题进行下载，并放入到themes文件夹下。

四、本地启动博客

hugo server -t m10c --buildDrafts

m10c为下载主题的名字，在本地启动服务

可以在localhost:1313浏览目前的博客

五、部署博客到服务器

本人使用的是github提供的服务器

首先创建新的github仓库

我们创建的仓库的名字应该是这种格式"账户名.github.io"，比如我创建的仓库的名字为"godhorrible.github.io"

然后通过hugo对该域名进行关联

hugo --theme=maupassant --baseUrl="https://godhorrible.github.io/" --buildDrafts

运行后在myblog下会产生一个public文件夹

进入public文件夹将该目录关联到github

# 关联git仓库
git remote add origin https://github.com/godhorrible/godhorrible.github.io.git

# 提交文件
git add .
git commit -m "第一次提交"
git push

此时可访问https://godhorrible.github.io来访问自己的博客

六、写文章

通过markdown进行文章的编写，将.md文件放入content/post目录下即可
